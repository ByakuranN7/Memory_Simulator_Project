# MH Simulator

Simulador de hierarquia de memória baseado na arquitetura MIPS, desenvolvido em C.

O projeto realiza a simulação de mecanismos de memória utilizados em arquiteturas modernas, permitindo acompanhar a execução de instruções e o comportamento dos componentes internos do sistema em tempo real através de uma interface textual.

## Funcionalidades

- Execução de instruções MIPS carregadas de arquivo
- Simulação de registradores
- Memória principal
- Cache L1 e Cache L2
- TLB (Translation Lookaside Buffer)
- Tabela de páginas
- Write Buffer
- Tradução de endereços virtuais para físicos
- Política de substituição LRU
- Modos write-through e write-back
- Monitoramento de hits/misses das caches e TLB
- Execução passo a passo por ciclo

## Tecnologias

- Linguagem C
- Estruturas de dados
- Manipulação de memória
- Arquitetura de Computadores

## Objetivo

Projeto acadêmico desenvolvido para demonstrar o funcionamento interno de mecanismos de gerenciamento de memória e hierarquia de caches em sistemas baseados na arquitetura MIPS.
